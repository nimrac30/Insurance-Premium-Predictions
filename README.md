Project Summary
This project applies Python tools—Jupyter Notebook, scikit‑learn, pandas, and matplotlib—to build and evaluate regression models on a merged training/testing dataset.

Data Preparation
Merged raw training and test files into a unified dataset.

Applied custom missing‑value imputation tailored to feature types.

Feature Engineering
Created model‑ready features using:

Binary encoding

Ordinal encoding

Target encoding

Cyclical encoding for periodic variables

Exploratory Analysis
Visualized distributions, correlations, and outliers to uncover trends and data quality issues.

Modeling
Trained multiple regression algorithms.

Evaluated and compared performance using RMSE as the primary metric.
